# Sistema de puntuación

El sistema de puntuación se calculará semanalmente.

## Ejemplos de eventos positivos

- Salvó la noche
- Frase mítica
- Mejor outfit
- Ligue confirmado
- Invitó a una ronda
- Hizo reír al grupo

## Ejemplos de eventos negativos

- Desapareció sin avisar
- Llegó tarde
- Se durmió en mitad del plan
- Momento incómodo
- Prometió salir y se borró

## Reglas

- Cada liga podrá personalizar sus eventos.
- Los eventos podrán sumar o restar puntos.
- Los eventos deberán ser aprobados por un administrador.
- Los jugadores podrán pedir eliminar eventos sensibles.
